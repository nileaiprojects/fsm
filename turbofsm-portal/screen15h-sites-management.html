📝 Sites Management (Screen 15h):
  • Site Entity: First-class entity linked to a Customer/Lead. Site ID format: SITE-YYYYMMDD-NNNN. Each customer can have multiple sites.
  • Site vs. Billing Address: The customer's primary/billing address is stored on the Lead/Customer record. Sites are separate service locations where work is performed.
  • Site Fields: Site Name (required), Address (required), City, State, Zip, Google Location Pin, Site Contact Name, Site Contact Phone, Site Notes (access codes, gate info), Active/Inactive flag, Primary flag.
  • Usage: Sites appear as dropdowns on Appointment (Screen 8a), Estimate (Screen 11), and Job (Screen 13a) forms. One site = one job. An estimate can cover multiple sites, spawning multiple jobs on approval.
  • Inline Add: Sites can be added from any screen via "+ Add New Site" button, which links back to the customer record.
